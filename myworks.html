Coursera Offline

This is a script to download course videos and subtitles from Coursera in one go without having to press several links.
It has features like:

  • Week Structure: The script takes care of organizing the videos into separate folders for separate weeks.
  • Synching: So new videos have been added into the course. Don't worry the script takes care of synching with Coursera. All you need to do is run a simple command.
  • Ordering the videos: So you've downloaded all the videos. Now you might be wondering the order in which the videos are meant to be watched. Don't worry ! The script renames the videos so that the names of videos that come first are lexicographically smaller than those that come later.
  • VLC Compatibility: The script downloads the subtitle for each video into a Subs folder so that vlc automatically loads the subs while playing the video.
  • Lightning fast download: The script downloads the videos parallely there by making the overall download pretty fast.
You can get the source code from GitHub or PyPi. You can find the installation instructions here.
